# 신창섭 키우기: 정상화의 길

이 게임은 GitHub Pages를 통해 배포되는 Idle Clicker 게임입니다.

## 접속 방법
1. 이 저장소의 **Settings > Pages** 탭으로 이동합니다.
2. **Build and deployment > Source**를 "Deploy from a branch"로 설정합니다.
3. **Branch**를 "main"으로 선택하고 저장합니다.
4. 잠시 후 상단에 생성되는 URL로 접속하여 게임을 즐기실 수 있습니다.

## 주요 기능
- 클릭을 통한 포인트 획득
- 다양한 상점 아이템 및 업그레이드
- 실시간 랭킹 시스템 (백엔드 연동)
- 배경음악 및 시각 효과
